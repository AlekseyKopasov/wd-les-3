Скрипты

  • Java, C#, Golang, Perl, Ruby – если логика формируется на стороне сервера

Скрипты

  • Java, C#, Golang, Perl, Ruby – если логика формируется на стороне сервера
  • WebAssembly (или Wasm) — позволяет переносить код, написанный на таких языках как C, C++, C#, Rust, в низкоуровневые ассемблерные инструкции и использовать его в сети. Формат имеет компактные размеры, высокую производительность, близкую к нативной, и может одновременно работать с JavaScript

Библиотеки и фреймворки

  • Библиотека - набор, правил, инструкций готовых решений

  • Фреймворк - это набор низкоуровневых библиотек, который позволяет пользователю создавать приложения. Он управляет всеми вызовами библиотек

Библиотеки

  • JS – JQuery, React.js
  • CSS – Animate.css, Normalize.css

Фреймворки

  • JS – Angular, Vue.js, Next.js
  • CSS – Tailwind CSS, Bootstrap, Materialize CSS

CMS и CRM

  • CMS - это панель управления сайта, с помощью нее можно редактировать контент (товары, статьи) без знания программирования

    WordPress, Joomla, OpenCart, 1С-Битрикс, MODx, Drupal, PrestaShop

CMS и CRM
