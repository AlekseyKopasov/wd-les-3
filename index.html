Скрипты

  • Java, C#, Golang, Perl, Ruby – если логика формируется на стороне сервера
  • WebAssembly (или Wasm) — позволяет переносить код, написанный на таких языках как C, C++, C#, Rust, в низкоуровневые ассемблерные инструкции и использовать его в сети. Формат имеет компактные размеры, высокую производительность, близкую к нативной, и может одновременно работать с JavaScript

Глобальные атрибуты

можно вставить в любой тег

accesskey, autocapitalize, autofocus, contenteditable, dir, draggable, enterkeyhint, hidden, inert, inputmode, is, itemid, itemprop, itemref, itemscope, itemtype, lang, nonce, popover, spellcheck, style, tabindex, title, translate

Глобальные атрибуты

Можно вставить в любой тег

accesskey, autocapitalize, autofocus, contenteditable, dir, draggable, enterkeyhint, hidden, inert, inputmode, is, itemid, itemprop, itemref, itemscope, itemtype, lang, nonce, popover, spellcheck, style, tabindex, title, translate

Атрибуты событий

  • Для объекта окна (применяются к тегу <body>): onload, onresize
  • События формы: onblur, onchange, oncontextmenu, onsubmit
  • События с помощью клавиатуры: onkeydown, onkeypress, onkeyup
  • События с помощью мыши: onclick, onmousedown, onmousemove, onwheel
  • События перетягивания: ondrag, ondragend, ondragenter, ondragleave, ondragover
  • События буфера обмена: oncopy, oncut, onpaste

Сокращенные атрибуты

Атрибуты указания пути

Атрибуты селекторы

Предназначены для создания выборки элементов (используются в CSS и JavaScript)

  • id="имяУникальногоИдентификатора"

Атрибут стиля

это атрибут, который размещает CSS-код непосредственно в HTML-тег

  • style="color: tomato;"

Атрибуты data-*

это пользовательский атрибут на HTML-элементе, название которого начинается с data-

  • data-id="features"

  • data-slide="current"
  • data-user="{id:1; role: 'admin';}"

Атрибуты тега select

  • autocomplete — подсказка для функции автозаполнения формы
  • disabled — делает элемент неактивным
  • form — связывает список с формой
  • multiple — позволяет выбрать несколько опций
  • name — задает имя элемента, которое будет отправляться на сервер
  • required — делает элемент обязательным для заполнения
  • size — задает количество строк в списке
  • value — для отправки значений на сервер
